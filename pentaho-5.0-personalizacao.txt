Edite a imagem:
puc-login-logo.png

Comentar o código abaixo:

<code>

<div id="eval-users-toggle-container">
            <%
              if (showUsers) {
            %>

                <div id="eval-arrow" class="closed"></div>
            </div>

            <%
            } else {
            %>
            &nbsp;
            <%
              }
            %>
</div>

</code>
